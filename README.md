#  TaskTrek – React Todo App

A simple and clean **Task Management / Todo App** built with **React 18**

##  Features

-  Add new tasks  
-  Add tags with tasks  
-  Tasks automatically saved in LocalStorage  
-  Tasks categorized by status (To do / Doing / Done)  
-  Delete tasks  
-  Clean UI & responsive design  

---



## Installation

```
git clone <https://github.com/mrcoder569mustafa-cpu/Todo_Task.git>
cd project-folder
npm install
npm start
```

---

##  Tech Stack

- React 18
- CSS
- LocalStorage
- Vite / CRA
